# haframjolk.com

This repository contains the code for my website, [https://haframjolk.com](haframjolk.com).

## Usage

### Setup

```sh
git clone git@github.com:haframjolk/haframjolk.com.git
cd haframjolk.com
npm install
```

### Developing

```sh
npm start
```

### Testing

```sh
npm test
```

### Building

```sh
npm run build
```

### Deploying

```sh
npm run deploy
```
